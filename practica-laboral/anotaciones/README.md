## 🧠 Metodología documentada

El archivo cubre el flujo de trabajo dividido en 5 etapas:

1. **Revisión diaria** — verificar agentes activos, alertas críticas/altas de las últimas 24h
2. **Investigación de alertas** — análisis por severidad (0–3 bajo / 4–7 medio / 8–10 alto / 11–15 crítico)
3. **Respuesta y acciones** — contención, escalamiento, documentación
4. **Gestión de agentes** — estado de conexión, configuración
5. **Reportes y mejoras** — generación de informes, ajuste de reglas

---

> 💡 Estas notas reflejan la metodología real aplicada durante la práctica como analista SOC en entorno productivo con Wazuh.
